# ML4E-Induction-Task

This project is a simple and responsive website developed for the ML4E(Machine Learning For Everyone)  as part of the club induction process.
The purpose of this website is to provide participants with all the essential information about the hackathon in a clean and organized manner.
The website is designed using HTML and CSS focusing on simplicity,clarity,and smooth user experience.It includes basic web design features such as navigation menus,hover effects,transitions,and responsive layout,making it easy for users to explore the hackathon details.
FEATURES :
1. Clean and minimal design
2. Navigation bar for easy page access
3. Hover effects and smooth transitions
4. Organized multi-page structure
5. Suitable for hackathon or event promotion.
The project highlights front-end development fundamentals,including layout design,styling,and structured content presentation.
